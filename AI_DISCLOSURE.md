## 2. Purpose and Scope of Use

AI tools were employed in the following specific aspects of the project:

- **Initial code scaffolding:** Generating foundational Streamlit dashboard components and boilerplate UI structure.
- **Data preprocessing:** Aiding in writing Python scripts for simulation data generation and CSV export.
- **Debugging:** Suggesting fixes for error messages and resolving syntax issues in Python/SQL code.
- **Syntax explanation:** Explaining unfamiliar library functions (e.g., scikit-learn, pandas, matplotlib) to accelerate team learning.
- **Documentation assistance:** Providing structural suggestions for README and proposal formatting (not content generation).

---

## 3. Methodology of Use

AI outputs were obtained through carefully designed prompts, typically structured as:

> *"I need a Streamlit component that displays a heatmap from a pandas DataFrame. Please provide the code and explain how it works."*

All generated content—including but not limited to code snippets, UI component suggestions, and debugging recommendations—was **strictly reviewed, tested, and substantively modified** by the team member who integrated it. No AI-generated code was adopted without:

- **Manual testing** in the development environment
- **Verification** against the project's data schema and logic
- **Full comprehension** by the responsible team member (per the course's "explain-back" rule)
